# Markdown 渲染测试内容

安装完依赖后，你可以在聊天中发送以下消息来测试 Markdown 渲染效果：

---

## 测试消息 1: 基础格式

```
请用 Markdown 格式回复我：

# 这是一级标题
## 这是二级标题

这是一段普通文本，包含 **粗体** 和 *斜体* 还有 ~~删除线~~。

这是一个 `行内代码` 示例。
```

---

## 测试消息 2: 代码块

```
给我写一个 TypeScript 函数示例
```

AI 可能会回复类似：

````markdown
```typescript
function greet(name: string): string {
  return `Hello, ${name}!`;
}

const message = greet("World");
console.log(message);
```
````

你会看到带语法高亮的代码块！

---

## 测试消息 3: 列表

```
列出学习 React 的步骤
```

AI 可能会回复：

```markdown
学习 React 的步骤：

1. **基础知识**
   - HTML/CSS/JavaScript
   - ES6+ 语法
   - npm/yarn 包管理

2. **核心概念**
   - 组件和 Props
   - State 和生命周期
   - Hooks (useState, useEffect)

3. **进阶内容**
   - Context API
   - 路由 (React Router)
   - 状态管理 (Redux/Zustand)
```


## 📝 快速测试命令

最简单的测试方式：

**发送**: `用 Markdown 格式给我介绍一下 React Hooks`

你会看到包含标题、代码块、列表等多种格式的回复！

---

## ✨ 预期效果

安装完成后，你会看到：

1. ✅ **标题** - 不同大小和粗细
2. ✅ **代码高亮** - 彩色语法，带行号
3. ✅ **表格** - 格式化表格，支持横向滚动
4. ✅ **列表** - 有序/无序列表，支持嵌套
5. ✅ **链接** - 蓝色可点击，自动打开新窗口
6. ✅ **引用** - 带左侧边框和背景色
7. ✅ **任务列表** - 可视化复选框
8. ✅ **流畅动画** - 消息出现时的淡入效果
